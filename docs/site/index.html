Hardware Layer

2√ó2 TPU bladder array under sternum plate + pressure sensor(s) + ADC. Hall-effect sensor and magnet align with sternum movement for depth tracking. Sensors measure compression force, depth, and placement.

Firmware Layer

Real-time sensor reading and event detection. Computes: compression count, rate (cpm), depth estimate (mm), pause time, and placement drift (left/right/up/down). Data processed locally and sent to dashboard.

Dashboard Layer

Live graphs and performance indicators. Session management with automatic save. Score summary and instructor comments box. Exportable reports (CSV/PDF) for documentation and student feedback.

Calibration System

Zero calibration at rest, depth calibration with known displacement steps, pressure calibration per bladder, and threshold tuning for adult/child profiles.
