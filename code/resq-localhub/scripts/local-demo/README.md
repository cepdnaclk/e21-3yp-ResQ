# Local Demo Scripts

These scripts are the Windows-friendly helpers for Phase 11 local demo runs.

- `check-local-demo-health.ps1` checks backend health, service-info, broker reachability, and demo setup details.
- `start-firmware-simulator.ps1` starts the local firmware simulator with a chosen device ID and behavior flags.
- `demo-mqtt-watch.ps1` wraps the MQTT trace helper for demo usage.

Run them from the repository root unless the script notes otherwise.
